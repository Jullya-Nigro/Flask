## Sobre o Projeto

Este projeto é uma **API RESTful simples** desenvolvida em **Python com Flask**.  
Ele permite criar, listar, atualizar e deletar usuários em memória, sendo ideal para **estudo de CRUD e APIs**.  

O projeto utiliza:
- Python 3.x
- Flask
- JSON para comunicação

---

## Funcionalidades

- ** Listar usuários** → `GET /users`
- ** Criar usuário** → `POST /users/`
- ** Atualizar usuário** → `PUT /users/<id>`
- ** Deletar usuário** → `DELETE /users/<id>`
